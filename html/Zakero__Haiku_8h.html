How To Use It?

Step 0

Your compiler must support at least the C++20 standard. The location of the Zakero_Haiku.h header files must be in your compiler's include path.

Step 1

The first step is to select which C++ source code file will contain the Zakero Haiku implementation. Once the location has been determined, add the following to that file:

#define ZAKERO_HAIKU_IMPLEMENTATION
#include "Zakero_Haiku.h"

The macro ZAKERO_HAIKU_IMPLEMENTATION tells the header file to include the implementation of the Haiku.

In all other files that will use the Haiku, they only need to include the header.

#include "Zakero_Haiku.h"
